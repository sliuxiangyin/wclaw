# 微信桥专项实施计划（v3）

目标：在**不增加宿主微信专用路由**的前提下，完成「chat 内 `/login` → 多账号 → 通用会话切换」。

---

## 原则

1. 宿主只暴露通用 API：`/api/plugins/:pluginId/chat`、`/sessions`、`/sessions/:sessionId/switch`。
2. 登录与账号语义全部由插件运行时（`plugin.json` 的 `entry`）实现，宿主通过**动态加载**调用 `handleChat` / `decorateSessions`，禁止 `if (pluginId === 'weixin-bridge')`。
3. 多账号 = 多个 `sessionId`（插件可往新 session 写入首条消息以便列表可见）。

---
