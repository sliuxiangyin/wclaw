# 国产显卡砺算7G100将于5月20日在京东发售

国产GPU领域再传捷报！砺算科技（LISUAN）正式宣布，其首款消费级显卡 **7G100** 将于 **5月20日** 在京东平台首发开售。这款备受关注的国产显卡，凭借其宣称的强劲性能和亲民定位，有望为沉寂已久的显卡市场带来一股新风。

## 核心规格速览

砺算7G100定位为消费级显卡，其核心参数如下：

- **制造工艺**：采用先进的 **6nm** 制程工艺。
- **显存配置**：搭载 **12GB** 大容量显存。
- **性能表现**：官方宣称游戏性能对标 **NVIDIA GeForce RTX 4060**。
- **系统认证**：已正式通过 **微软WHQL认证**，确保与Windows系统的兼容性与稳定性。

## 实机性能：挑战《黑神话：悟空》

砺算科技官方已公布了一段使用7G100运行《黑神话：悟空》的实机演示视频，展示了其在热门3A大作中的实际表现：

- **画质设置**：4K分辨率，超采样清晰度设置为41，高画质预设，并锁定60帧上限。
- **帧率表现**：在测试场景中，帧率在 **50-100帧** 之间浮动，整体流畅度可玩。
- **显存占用**：显存占用率在 **60%-90%** 之间，表明12GB显存能较好地应对高画质下的资源需求。

## 市场意义与玩家期待

砺算7G100的发布，被许多玩家视为打破NVIDIA和AMD在消费级显卡市场垄断格局的重要一步。其核心看点在于：

- **国产替代新选择**：为追求自主可控硬件的用户提供了新的选项。
- **价格竞争预期**：若其定价合理且性能达标，有望对主流显卡市场形成压力，推动整体价格回归理性。
- **生态逐步完善**：通过WHQL认证和《黑神话：悟空》等热门游戏的实机演示，表明其驱动和软件生态正在快速成熟。

## 发售信息

- **开售平台**：京东商城
- **开售日期**：2024年5月20日

目前官方尚未公布最终售价，但考虑到其定位和竞品价格，市场普遍期待一个具有竞争力的定价。对于关注国产硬件发展、或希望以更实惠价格体验主流游戏的玩家来说，砺算7G100无疑值得密切关注。
