# redis之父给deepseek-v4-flash单独写了个C版本的推理引擎，能且只能跑在MacOS上

> 作者：morry_wang

## 事件概述

Redis 之父 antirez 最近在 GitHub 上开源了一个名为 **ds4** 的项目，这是一个专门为 DeepSeek 4 Flash 模型编写的 C 语言推理引擎。有趣的是，这个引擎**目前仅支持在 MacOS 上运行**，利用了 Apple 的 Metal 框架进行加速。

## 项目亮点

- **纯 C 语言实现**：没有依赖 Python 或其他高级语言框架，性能更可控。
- **专为 Metal 优化**：充分利用 Mac 的 GPU 能力进行推理加速。
- **轻量级设计**：专注于 DeepSeek 4 Flash 模型的本地推理。

## 技术细节

从项目描述来看，antirez 选择用 C 语言重写推理引擎，并绑定到 Metal 框架，而不是使用更常见的 CUDA 或 Vulkan。这意味着：

- 目前仅支持 **Apple Silicon** 或 **AMD GPU** 的 Mac 设备。
- 无法在 Windows/Linux 或 NVIDIA GPU 上运行。
- 适合 Mac 开发者进行本地实验和测试。

## 社区反应

该项目在 GitHub 上已经引起了不少关注，开发者们对 antirez 的“硬核”做法表示赞赏。有人调侃说：“这是为了让 Mac 用户也能玩上 DeepSeek 吗？”

## 如何体验

如果你有一台 Mac，可以尝试以下步骤：

1. 克隆仓库：`git clone https://github.com/antirez/ds4.git`
2. 按照 README 中的说明编译。
3. 准备好 DeepSeek 4 Flash 模型文件。
4. 运行推理引擎进行测试。

## 总结

antirez 的这个项目展示了即使是最新的大模型推理，也可以用最基础的语言和框架来实现。虽然目前只支持 MacOS，但或许未来会扩展到其他平台。对于 Mac 用户来说，这是一个直接在本地运行 DeepSeek 4 Flash 的好机会。
