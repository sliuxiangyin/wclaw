# DeepSeek 大范围开放“识图模式”，正式跨入图文交互时代

近日，DeepSeek 迎来重大更新，开始大范围灰度测试“识图模式”，标志着其正式跨入图文交互时代。用户现在可以直接上传图片，让 AI 理解图像内容并进行对话。

## 什么是“识图模式”？

简单来说，DeepSeek 的“识图模式”允许用户上传图片文件，AI 能够识别图片中的文字、物体、场景等信息，并基于这些信息与用户进行交互。这从纯文本对话升级为多模态理解，极大拓展了应用场景。

## 实际应用场景

“识图模式”带来了丰富的实用场景，以下是几个典型用例：

- **文档与截图识别**：上传包含文字的截图或文档图片，AI 可提取并理解其中的文字内容。
- **物体与场景识别**：拍摄或上传物品照片，AI 能识别物体名称、类别，甚至提供相关信息。
- **图表与数据解读**：上传表格、流程图等，AI 可分析数据关系并给出解读。
- **视觉问答**：基于图片内容提问，例如“这张图里有什么？”“这个标志是什么意思？”

## 灰度测试与开放范围

目前，“识图模式”处于大范围灰度测试阶段，并非所有用户都能立即体验。DeepSeek 正在逐步向更多用户开放此功能。如果你尚未在界面中看到图片上传入口，可以稍后再试，或关注官方后续公告。

## 总结

DeepSeek 的“识图模式”是一次重要的功能升级，从纯文本到图文交互，让 AI 助手更贴近真实世界的交流方式。无论是办公文档处理、学习资料分析，还是日常生活中的视觉信息查询，这一功能都将带来更高效、更直观的体验。
