# 佬友们，0成本开站的方法找到了 —— Hugging Face Spaces

最近看到不少朋友在讨论开站成本，动辄几百上千，难道只有我还在坚持0元开站吗？😅 今天就把这个压箱底的方法分享给大家，**完全免费，零成本拥有自己的站点**。

## 第一步：注册 Hugging Face 账号

首先，访问 [Hugging Face 注册页面](https://huggingface.co/join) 创建一个账号。这是整个流程的基础，完全免费。

## 第二步：找到现成模板，一键复制

注册登录后，打开这个已经配置好的模板空间：
[New API - a Hugging Face Space by CHYbenzun666](https://huggingface.co/spaces/CHYbenzun666/New-API)

在页面右上角找到 **三个点** 的菜单按钮，点击它。

![](https://cdn3.ldstatic.com/optimized/4X/c/4/c/c4c1d714847c7e11250fa6695fe539fbdbdafd3b_2_690x310.png)

## 第三步：复制空间（Duplicate）

点击三个点后，选择 **Duplicate this Space** 选项。

![](https://cdn3.ldstatic.com/original/4X/f/6/7/f6707707f3d2e708527d9876ae3def1c56977d09.png)

## 第四步：配置空间名称

在弹出的窗口中，给你的新空间取一个喜欢的名字，比如 `my-awesome-site`。

![](https://cdn3.ldstatic.com/original/4X/5/6/a/56a0722a864d03b944713a4ae93e19985ab90d1e.png)

## 第五步：设置环境变量（关键步骤）

继续往下滑，找到 **环境变量 (Secrets)** 设置区域。这里有几个关键变量需要配置：

![](https://cdn3.ldstatic.com/original/4X/2/9/8/2987b544e682dec1ed94ac0c7dde191981a030a3.png)

需要设置的变量包括：

- **SQL_DSN**：数据库连接字符串（如果你不需要数据库，可以留空或使用默认值）
- **CRYPTO_SECRET**：用于加密会话（session）的密钥，随便输入一串复杂字符即可
- **CRYPTO_SECRET**（注意有两个同名变量，但用途不同）：登录/注册加密密钥，同样随便输入复杂字符

> **提示**：变量值最好复杂一些，包含大小写字母、数字和特殊符号，确保安全。

## 第六步：创建空间

配置完成后，点击 **Create Space** 按钮。

![](https://cdn3.ldstatic.com/original/4X/a/4/4/a44425c95f8a6dcff4976361af2d06dd31f6aea5.png)

## 第七步：获取站点地址

等待空间创建完成（通常需要几十秒到几分钟），再次点击右上角的 **三个点**，选择 **Embed this Space**。

![](https://cdn3.ldstatic.com/original/4X/f/d/c/fdcee9c671c352ee61280bf0ac9400c3074a8647.png)

在弹出的窗口中，**Direct URL** 就是你的站点地址！复制它，直接访问即可。
