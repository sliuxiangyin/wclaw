# Google AI 驱动药物研发公司Isomorphic Labs，将筹集20亿美元，目标治愈所有疾病

据知情人士透露，由Alphabet旗下谷歌DeepMind分拆出来的AI驱动药物研发公司Isomorphic Labs正进行深入谈判，计划在新一轮融资中筹集超过20亿美元。这一消息引发了业界对AI在生命科学领域应用的广泛关注。

## 融资详情

- **主导方**：曾主导Isomorphic Labs去年首轮融资的风投机构Thrive Capital将牵头本轮新融资。
- **参与方**：Alphabet也将参与，但融资尚未最终完成。
- **金额目标**：超过20亿美元。

## 公司背景与愿景

Isomorphic Labs致力于将AI进展应用于加速药物发现并降低研发成本。其核心目标是通过AI技术，从根本上改变药物研发的效率和成功率，最终实现“治愈所有疾病”的宏大愿景。

## 行业趋势与竞争

- **AI制药热潮**：做出此类尝试的公司越来越多，尽管硅谷在进军医疗和生命科学方面屡遭失败。
- **竞争对手**：谷歌竞争对手、同样获得Thrive支持的OpenAI，近期也推出了一款旨在加快药物研发的AI模型。

## Alphabet的战略布局

这笔新增融资将是Alphabet长期以来将其前沿押注转化为独立业务计划的又一步。此前，Alphabet的自动驾驶子公司Waymo今年早些时候以1260亿美元的估值融资160亿美元，显示出Alphabet在推动前沿技术商业化方面的决心。

## 未来展望

随着AI技术的不断成熟，其在药物研发领域的应用前景备受期待。Isomorphic Labs此次大规模融资，不仅将为公司提供充足的研发资金，也将进一步推动AI制药行业的发展。我们拭目以待，看AI技术如何改变医疗健康的未来。
